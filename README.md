Data Analytics Project
Overview

This project demonstrates an end-to-end data analytics workflow starting from data loading and exploration to dashboard creation and reporting. The goal is to extract meaningful insights from the dataset using Python, SQL, and Power BI, and present the findings through a professional dashboard, report, and presentation.

The project covers:

Data loading and preprocessing

Exploratory Data Analysis (EDA)

Data cleaning and transformation

SQL analysis using relational databases

Dashboard development in Power BI

Final insights presentation
